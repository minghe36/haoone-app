## Activation Steps

### Step 1: Open Activation Dialog

1. Launch the haoone application and complete user login
2. Check the top-right corner of the application interface
3. Find the **Activate** button or menu option
4. Click to open the activation dialog

**Indicators**:
- If "Not Activated" or activation countdown is displayed, activation is required
- If activation code countdown is shown (e.g., "5 days remaining"), trial period is ending soon

![Activation Dialog Example #w300](../image/iShot_2026-01-08_14.07.21.png)

### Step 2: Enter Activation Code

In the activation dialog, you have two ways to obtain an activation code:

#### Method A: Manual Input

1. Find the **Activation Code Input Field**
2. Enter or paste your activation code

#### Method B: Select from Purchased Codes

1. Click the **Select Activation Code** dropdown menu
2. View all your purchased activation codes
3. Click the activation code you want to use
4. The system automatically fills in the activation code field

### Step 3: Confirm Activation

After clicking confirm, you will see the activation interface:

![Activation Dialog Example #w300](../image/iShot_2026-01-08_14.47.41.png)

During activation, the system automatically records:

| Information | Description |
|------|------|
| **Device ID** | Unique identifier of the computer |
| **Hardware Fingerprint** | Identifier generated based on hardware features |
| **Operating System** | macOS, Windows, or Linux |
| **Activation Date** | Specific date and time of activation |
| **Device Name** | Computer name |

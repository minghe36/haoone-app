title: "6 Create, Rename, Delete, Switch Projects"
---

In the top-left corner of the software, you can manage projects including Create, Rename, Delete, and Switch Projects.

Transcription results from DaVinci Resolve/PR plugins are placed in the currently open project by default.

![#w300](../image/iShot_2026-01-09_11.52.04.png)

Project names are constructed using dates.

## Create Project

Click the New button

## Switch Project

You can switch between different projects using the "List" button

## Open Project

Opening a project opens the project folder, where you can view transcription results.

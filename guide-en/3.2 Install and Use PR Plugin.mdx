title: "3.2 Install and Use PR Plugin"
---

haoone provides a PR integration plugin that enables you to perform one-click transcription directly within Premiere Pro. No need to export audio or switch applications to complete the entire transcription workflow.

Note: Currently on Windows, the haoone GUI interface may appear (unresponsive) when executing the plugin. This is normal; manually switch back to the PR interface.

## Manually Enable Debug Mode

If you have AI coding tools like Claude Code, you can ask them to help you set this up.

### Mac System

Open Terminal and run the following command:

```bash
# Write debug mode setting
defaults write com.adobe.PrPro "PlayerDebugMode" -bool true
```

### Windows System


1. Press `Win + R` to open the Run dialog
2. Type `regedit` and press Enter to open Registry Editor
3. Navigate to: `HKEY_CURRENT_USER\Software\Adobe\Premiere Pro\Settings`
4. Right-click in the blank area on the right → New → DWORD Value
5. Name it: `PlayerDebugMode`
6. Double-click to set the value data to: `1`

## haoone Software Introduction

Covers all core needs of the subtitle editing and subtitle translation workflow, possibly the most comprehensive subtitle software on the market.

haoone software supports permanent free use of basic features, with one-time purchase for enhanced features, no subscription fees. Abandons the whisper solution, deeply optimized based on qwen3-asr, high Chinese recognition accuracy, transcription engine based on rust+c++, fast transcription speed, automatic GPU acceleration. Self-developed word-level audio alignment.
Chinese-English recognition accuracy and stability, word-level alignment precision surpass all whisper wrapper software on the market.

Local transcription, remote transcription, manuscript matching, smart line splitting, AI correction, AI smart hot words, translation, bilingual subtitles, professional subtitle editor, subtitle burning, custom LLM API...

Let tools adapt to your workflow: DaVinci Resolve subtitle plugin, PR subtitle plugin, command line tool, skill integration

One-click batch operations for multiple files

Long audio/video optimization: Can transcribe 2-hour files (Mac takes only 10 minutes)

Fast: Local transcription of 3-minute video completes within 20 seconds on Mac mini M4 16G

Accurate: Over 96% Chinese-English transcription recognition accuracy, with smart hot words that improve with use

Complete: Self-developed word-level alignment algorithm, 98% word-audio alignment rate

Supports both Windows and Mac platforms

## Feature List

haoone is a professional subtitle software with powerful capabilities. Core features:

* Local Transcription: Deeply optimized based on qwen3-asr, supports multiple languages, self-developed word-level alignment algorithm
* Remote Transcription: Fast, accurate, and complete remote transcription service
* Model Download: One-click download, also supports cloud storage download
* AI Smart Line Splitting
* AI Hot Word Replacement
* AI Correction
* DaVinci Resolve Subtitle Plugin
* PR Subtitle Plugin
* haoone-cli Command Line Tool
* Subtitle Skill
* Manuscript Matching: No word limit, supports automatic manuscript formatting, dual line splitting modes
* Subtitle Burning
* Bilingual Subtitles
* Professional Subtitle Editor
* Multi-view: Quick edit view, chapter view
* Translation
* One-click Batch Operations
* Search and Batch Replace
* ...

## 💻 系统要求

### macOS

- **操作系统**: macOS 10.15 (Catalina) 或更高版本
- **处理器**: Apple Silicon (M2/M3/M4)
- **内存**: 至少 8GB RAM
- **存储空间**: 至少 500MB 可用空间

### Windows

重要：windows 需要独立显卡，不然会无法执行或执行很慢

- **操作系统**: Windows 10 或 Windows 11（64位）
- **处理器**: Intel Core i5 或更高
- **内存**: 至少 8GB RAM
- **显卡**: 需要独立显卡
- **存储空间**: 至少 500MB 可用空间

### 步骤 1: 下载安装包

1. 访问官网下载页面：[https://haoai.pro/haoone/download](https://haoai.pro/haoone/download)
2. 点击 **macOS 下载** 按钮

1. **打开安装包**

3. **首次打开**
   - 请允许软件的我的文档文件夹访问权限

4. **完成用户登录**

## 软件更新

haoone 配备了内置的自动更新系统，可帮助您保持软件始终处于最新状态。系统会自动检查更新，并通知您有关可用的新版本。

二种方法，优先使用第一种方法：

* 1 有新版本后你会在软件上部看到更新通知，点击“下载并安装更新”即可，一般 3 分钟即可完成下载，安装成功后，软件会自动重启
* 2 到官网下载最新的安装包安装[https://haoai.pro/haoone/download](https://haoai.pro/haoone/download)，需要重新登录

### 更新会删除我的项目吗？

不会！更新过程是**完全安全的**：

✅ 所有项目文件保持不变
✅ 个人设置和偏好保留
✅ 激活信息保持有效
✅ 不会丢失任何数据

更新只替换应用程序本身，不涉及您的任何数据。

### 更新过程中关闭应用会怎样？

避免在更新过程中关闭应用。

### 可以回滚到旧版本吗？

可以，到官网下载历史版本安装：[https://haoai.pro/haoone/download](https://haoai.pro/haoone/download)

### 自动更新后需要重新下载模型吗？

不用重新下载。

### 安装过程中出现"权限被拒绝"

**症状**：
- 显示 "Permission Denied" 或 "访问被拒绝"
- macOS 安装失败

**原因**：
- 用户权限不足
- 磁盘被锁定
- 安全软件阻止安装

**解决步骤**：

**macOS**：
1. 检查管理员权限：
   - 系统偏好设置 > 用户与群组
   - 确认当前用户有管理员权限

2. 关闭安全软件：
   - 暂时禁用防病毒软件
   - 有些软件会阻止应用安装

3. 手动安装：
   - 从官网下载最新版本
   - 手动替换应用程序文件夹中的应用

**Windows**：
1. 以管理员身份运行：
   - 右键点击安装程序
   - 选择 "以管理员身份运行"

2. 禁用防病毒软件：
   - 暂时关闭 Windows Defender
   - 关闭第三方防病毒软件

3. 清理文件锁定：
   - 关闭所有 haoone 进程
   - 重启计算机后重试

### 问题：更新后应用无法打开

**症状**：
- 更新完成后，应用不能启动
- 每次点击都无反应
- 应用立即崩溃

**原因**：
- 安装过程中中断
- 磁盘空间不足
- 系统库缺失

**解决步骤**：
1. 确保有足够的磁盘空间：
   - macOS：至少需要 1 GB 可用空间
   - Windows：至少需要 500 MB 可用空间
   - 清理垃圾文件和临时文件

2. 检查应用完整性：
   - 重启计算机
   - 重新启动应用

3. 重新安装：
   - 卸载 haoone
   - 从官网重新下载并安装

4. 重置应用偏好：
   - macOS：
     ```bash
     rm -rf ~/Library/Application\ Support/com.haoone
     ```
   - Windows：删除 `C:\Users\<用户名>\AppData\Local\com.haoone`
   - 重新启动应用

## 卸载应用

如果您想要卸载 haoone：

#### 第一步：可选 - 解绑激活码

通过官方网站可以解绑设备

#### 第二步：卸载应用

**macOS**：
1. 打开 Finder
2. 进入应用程序文件夹
3. 找到 haoone
4. 右键点击 → 移至废纸篓
5. 清空废纸篓

**Windows**：
1. 打开 控制面板
2. 进入 程序 → 程序和功能
3. 找到 haoone
4. 点击卸载
5. 按照提示完成卸载


#### 第三步：清理残留文件（可选）

删除我的文档下的 haoone 目录。

**macOS**：
```bash
# 删除应用配置和数据
rm -rf ~/Library/Application\ Support/com.haoone
rm -rf ~/Library/Caches/com.haoone
```

**Windows**：
```
删除文件夹：C:\Users\<用户名>\AppData\Local\com.haoone
```

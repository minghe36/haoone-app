haoone 软件是强大的字幕编辑器，能够帮助你快速、高效地编辑和调整转录字幕。

软件会自动保存修改结果，没有保存按钮。

<video src="https://cdn.haoai.pro/assets/hao-one/6.mp4" controls />



### 快速拆行

在想要拆行的位置，按 enter 键，就会以光标为前后边界拆行。

![#w600](../image/iShot_2026-01-09_12.18.47.png)

### 上下键

上下键可以将光标聚焦到上一个字幕或下一个字幕。

###  字幕后四个按钮操作说明

![#w300](../image/iShot_2026-01-09_12.22.37.png)

删除、合并到上一行、合并到下一行、添加一行

### 修改后如何同步到达芬奇？

很简单，打开 haoone 插件，点击“同步软件的字幕修改”即可：

![#w600](../image/iShot_2026-01-09_12.16.13.png)

### 键盘快捷键

| 快捷键 | 功能 | 平台 |
|--------|------|------|
| Enter | 拆分当前行 | Windows / macOS |
| Ctrl+, / Cmd+, | 向前合并行 | Windows / macOS |
| Ctrl+. / Cmd+. | 向后合并航 | Windows / macOS |
| Ctrl+- / Cmd+- | 删除当前字幕 | Windows / macOS |
| Ctrl++ / Cmd++ | 新增一个字幕区块 | Windows / macOS |
| Ctrl+Z / Cmd+Z | 撤销 | Windows / macOS |
| Ctrl+Y / Cmd+Y | 重做 | Windows / macOS |
| Escape | 取消编辑 | Windows / macOS |
| ↑ / ↓ | 在字幕行间导航 | Windows / macOS |
